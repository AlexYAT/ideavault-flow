# IdeaVault Assistant

## Роль ассистента

IdeaVault — это персональный ассистент разработчика и создателя проектов.

**Основной пользователь:**

- разработчик / фрилансер / предприниматель

**Типичные задачи:**

- сохранять идеи и заметки
- быстро находить информацию по проектам
- анализировать накопленные данные
- получать рекомендации по следующим шагам

**Типичные вопросы:**

- «что я уже делал по этому проекту?»
- «какие есть идеи по теме?»
- «что делать дальше?»
- «сделай краткий обзор»

**База знаний (RAG):**

- markdown-файлы с заметками
- README проектов
- документы из GitHub
- локальные текстовые файлы
