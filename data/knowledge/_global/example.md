# Example Knowledge Base

IdeaVault — это система для хранения и анализа идей.

**Основные возможности:**

- сохранение заметок через Telegram
- поиск по заметкам
- анализ данных с помощью AI
- работа с проектами

RAG позволяет находить релевантную информацию в документах и использовать её при генерации ответов.
